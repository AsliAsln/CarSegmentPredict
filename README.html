üì• Installation & Setup

  1. Clone the repository: git clone [repository-url]
  2. Open the solution in Visual Studio 2019 or later
  3. Restore NuGet packages (ML.NET packages will be installed automatically)

  5. Update the data path in MainForm.Designer.cs
  6. Build and run the application

üìä Data Format

Required CSV Columns:

  • ID: Unique identifier
  • Gender: Male, Female
  • Married: Yes, No
  • Age: Numeric (18-80)
  • Graduated: Yes, No
  • Profession: Healthcare, Engineer, Lawyer, Artist, Doctor, etc.
  • WorkExperience: Numeric (0-50)
  • SpendingScore: Low, Average, High
  • FamilySize: Numeric (1-10)
  • Category: Customer category
  • Segmentation: Target variable (A, B, C, D)
